Interactive Editorial Web: Self-Reliance
Project Brief
랄프 왈도 에머슨의 "Self-Reliance"(1841) 텍스트를 인터랙티브 웹으로 구현한다. 이미지 없이 오직 타이포그래피, 레이아웃, 여백만으로 몰입감 있는 스크롤 경험을 만든다.

2. 색상 (이 값은 정확히 사용할 것)
순수한 흰색(#FFFFFF)과 검정(#000000) 대신 미세하게 틸트된 톤을 사용한다:
밝은 배경: #FAFAF8 (아주 살짝 따뜻한 화이트)
어두운 배경: #0A0A0C (아주 살짝 차가운 블랙)
밝은 쪽 중간톤: #E8E8E6
어두운 쪽 중간톤: #18181A
3. 타이포그래피 방향
폰트
제목/강조: Playfair Display (Google Fonts)
본문/캡션: Inter 또는 시스템 산세리프
느낌
전통적인 책 타이포가 아닌, 현대 웹에 최적화된 느낌
자간(letter-spacing)은 기본보다 타이트하게
단어 간격(word-spacing)은 기본보다 넓게
이 조합이 세련되고 공기감 있는 타이포를 만든다
크기 위계
히어로 타이틀: 화면을 압도할 만큼 크게. 데스크톱에서 뷰포트 폭의 상당 부분을 차지
섹션 타이틀: 충분히 크되, 여러 줄로 자연스럽게 떨어지는 크기
서브텍스트: 본문 수준. 읽기 편한 크기
캡션: 작고 절제된 느낌. 대문자로 처리
반응형
clamp()를 활용해 모바일~데스크톱까지 자연스럽게 스케일
모바일에서도 읽히되, 데스크톱에서 임팩트 있게
4. 섹션별 레이아웃과 연출
전체 흐름
code
Code
[정적/여백] 히어로 → [확장] 섹션1 → [전환] 섹션2 → [클라이맥스] 섹션3 → [여운] 아웃트로
배경 전환:
히어로 ~ 섹션1: 밝은 배경
섹션2: 스크롤에 따라 밝은 → 어두운 배경으로 그라디언트 전환
섹션3 ~ 아웃트로: 어두운 배경
히어로
레이아웃
화면 정중앙 배치
극단적인 여백. 텍스트 외에는 거의 비어있는 느낌
높이 100vh
연출
페이지 로드 시 타이틀이 먼저 페이드인
서브타이틀은 약간의 딜레이 후 페이드인
스크롤 시 위로 살짝 밀리면서 사라지는 패럴랙스
분위기
숨 고르기. 고요한 시작.
섹션 1: 빛
레이아웃
타이틀: 좌측 정렬, 화면 왼쪽에서 시작
서브: 우측 정렬, 타이틀과 대각선으로 대치되는 위치
둘 사이에 충분한 여백
연출
스크롤 진입 시 타이틀이 단어 단위로 순차 등장 (stagger 효과)
"빛줄기"라는 단어에서 자간이 잠시 벌어졌다 돌아오는 효과 (선택적)
서브텍스트는 타이틀 등장 완료 후 페이드인
분위기
텍스트 자체가 빛처럼 나타나는 느낌
섹션 2: 무시
레이아웃
상단: 타이틀이 전체 폭으로
하단: 서브텍스트 두 개가 좌우로 분할 배치 (데스크톱), 모바일에서는 세로 스택
연출
이 섹션에서 배경색이 밝은 톤에서 어두운 톤으로 전환
스크롤 위치에 연동된 그라디언트
텍스트 색상도 배경에 맞춰 반전
"무시해버린다" 부분이 흐렸다가 선명해지는 효과 (선택적)
분위기
분위기 전환점. 밝은 세계에서 어두운 세계로.
섹션 3: 선언
레이아웃
캡션: 상단 중앙, 작고 절제되게
타이틀: 화면을 가득 채우는 대형 텍스트. 줄바꿈 자연스럽게.
연출
스크롤에 따라 텍스트가 한 줄씩 등장
클라이맥스 구간. 가장 많은 텍스트가 가장 크게.
전체 문장이 완성되면 잠시 머무는 느낌
분위기
선언의 무게. 이 섹션이 감정적 정점.
아웃트로: 경고
레이아웃
타이틀: 상단 좌측
서브: 하단 우측
둘이 대각선으로 배치되어 긴장감
연출
최소한의 애니메이션. 거의 정적.
"부끄러운 상태가 된다" 부분만 이탤릭 또는 약간의 강조
분위기
여운. 질문을 던지고 끝.
5. 인터랙션 원칙
사용할 기법
Intersection Observer로 섹션 진입 감지
scroll 이벤트로 배경 전환 progress 계산
CSS transition/animation 활용
React의 useEffect, useRef, useState 활용
애니메이션 톤
과하지 않게. 텍스트가 주인공.
부드럽고 절제된 움직임
빠르기보다 우아하게
필수 효과
fade-in (기본 등장)
stagger (단어별 순차 등장)
parallax (히어로 스크롤 시)
background color transition (섹션2 배경 전환)
